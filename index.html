Beats Per Minute

This website is meant to give the user an outlet to tap into their emotions along with good music. It provides a clean minimal Interface with several buttons relating to several emotions. The user can click on the buttons to play music that matches the emotion. It will redirect the user to different Spotify playlists I created. The website consists of two pages, a "Home" page, and an "About Me" page. Both feature a navigation system that redirects users to any of the two pages. If Spotify is installed on your device, you can click on the buttons to play the music directly from the app. Otherwise, you will be redirected to the Spotify website to play the music.

Created through the use of HTML and CSS on Visual Studio Code.

Ice Cream Shop

This program is a virtual Ice Cream Shop that allows the user to purchase Ice Cream along with personal accommodations. It offers the ability to choose the amount of Ice Cream Cones and to change the amount of pounds of chocolate with text boxes. The interface features two checkboxes, one with the choice of a waffle cone and the other with the choice of double scoops. Each of the text turns green with an italic font when selected. The program also features a checkout screen that calculates the chocolate cost, the ice cream cost, and the total cost. The UI consists of big buttons to make the experience simple and easy for the user. The color scheme is designed to represent the flag colors of Mexico.

Created through the use of Java on Eclipse.
